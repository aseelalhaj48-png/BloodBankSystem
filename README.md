Blood Donation Management System
Project Description
Blood Donation Management System is a Java console-based application developed using Object-Oriented Programming (OOP) concepts. The system is designed to manage blood donors, patients, blood donations, and blood requests from hospitals in an organized and efficient way.
The application allows users to register donors and patients, record blood donations, manage blood requests, search for donors by blood type, and save donor information to a text file.


Main Features
- Register new donors
- Register new patients
- Record blood donations
- Add blood requests
- Display donors, patients, donations, and requests
- Search donors by blood type
- Save donor data to `donor

OOP Concepts Used
- Classes and Objects
- Encapsulation
- Inheritance
- Abstraction
- Polymorphism
- Method Overriding
- Composition
- Constructors




 Main Classes
- Main

- Patient

- Donation

 File Handling
The project saves donor information in a text file named `donors.txt` using `FileWriter` and `PrintWriter`.

 How to Run
1. Open the project in NetBeans or IntelliJ IDEA.
2. Run `Main.java`.
3. Choose an option from the menu and follow the instructions.
